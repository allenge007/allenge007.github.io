lec1: 仿射与凸性

仿射 (Affine Transformation)

仿射集（空间） (Affine Space)

说明
∀x1,x2∈C,{θx1+(1−θ)x2,∀θ∈R}⊆C\forall x_1, x_2\in C, \{\theta x_1 + (1 - \theta)x_2, \forall \theta \in R\} \subseteq C∀x1​,x2​∈C,{θx1​+(1−θ)x2​,∀θ∈R}⊆C

如何理解？

设想一个没有原点的向量空间，其中向量只有方向和大小。假设有甲乙两人，其中甲知道一个空间中真正的原点，但是乙认为另一个点 ppp 才是真正的原点。

现在求两个向量之和 a+ba + ba+b。乙画出 p→ap\to ap→a 和 p→bp\to bp→b，再用平行四边形找出二者之和 p+(a−p)+(b−p)p + (a - p) + (b - p)p+(a−p)+(b−p)

说明

凸组合

说明
x1,…,xn∈Cθ1,…,θk∈R∑θi=1,θi≥0∑θixix_1, \dots, x_n \in C \\ \theta_1, \dots, \theta_k \in R \\ \sum \theta_i = 1, \theta_i \ge 0 \\ \sum \theta_i x_ix1​,…,xn​∈Cθ1​,…,θk​∈R∑θi​=1,θi​≥0∑θi​xi​

凸包

说明
Conv C={θ1x1+,…,+θnxn}{\rm Conv\ } C = \{\theta_1x_1 + , \dots, + \theta_n x_n\}Conv C={θ1​x1​+,…,+θn​xn​}
示例
  1. 凸多边形是凸集。可以任取凸多边形内两点连线，证明其为凸集。
  2. 超平面：{x∣aTx=b,a≠0},a,b,x∈Rn\{x | a^Tx = b, a\neq 0\}, a, b, x\in R^n{x∣aTx=b,a=0},a,b,x∈Rn，可见，超平面将 RnR^nRn 空间分为了 2 部分。超平面显然是一个凸集，因为在该平面上任取 2 点连成的线段仍然在该平面内。
  3. 半空间：{x∣aTx≥b,a≠0}\{x | a^Tx \ge b, a\neq 0\}{x∣aTx≥b,a=0} 或 {x∣aTx≤b,a≠0}\{x | a^Tx \le b, a\neq 0\}{x∣aTx≤b,a=0}，这就是上述超平面将 RnR^nRn 分为的两个空间。
