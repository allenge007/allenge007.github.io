离散时间傅里叶变换

离散时间傅里叶变换是将离散时间序列从时域转换到频域的数学工具。它揭示了离散时间信号中包含的频率成分。与连续时间傅里叶变换不同，DTFT 的结果 X(ejω)X(e^{j\omega})X(ejω) 是一个关于连续频率变量 ω\omegaω 的周期函数，周期为 2π2\pi2π。

1. 非周期序列的 DTFT

对于一个非周期离散时间序列 x[n]x[n]x[n]，其离散时间傅里叶变换 X(ejω)X(e^{j\omega})X(ejω) 定义为：

X(ejω)=F{x[n]}=∑n=−∞+∞x[n]e−jωn(1)X(e^{j\omega}) = \mathcal{F}\{x[n]\} = \sum_{n = -\infty}^{+\infty} x[n]e^{-j\omega n} \quad \quad (1)X(ejω)=F{x[n]}=n=−∞∑+∞​x[n]e−jωn(1)

其中：

  • nnn 是离散时间整数索引。
  • ω\omegaω 是归一化数字角频率，单位是弧度/样本 (rad/sample\text{rad/sample}rad/sample)。它是一个连续变量。
  • ejωe^{j\omega}ejω 表示在复平面上的单位圆。
  • X(ejω)X(e^{j\omega})X(ejω) 通常是一个复数函数，表示信号在频率 ω\omegaω 处的幅度和相位。
    • 幅度谱：∣X(ejω)∣|X(e^{j\omega})|∣X(ejω)∣
    • 相位谱：∠X(ejω)\angle X(e^{j\omega})∠X(ejω)

周期性：X(ejω)X(e^{j\omega})X(ejω) 是以 2π2\pi2π 为周期的，即 X(ej(ω+2πk))=X(ejω)X(e^{j(\omega+2\pi k)}) = X(e^{j\omega})X(ej(ω+2πk))=X(ejω) 对于任意整数 kkk。因此，我们通常只分析主值区间，如 −π≤ω<π-\pi \le \omega < \pi−π≤ω<π 或 0≤ω<2π0 \le \omega < 2\pi0≤ω<2π。

离散时间傅里叶反变换 (Inverse DTFT, IDTFT) 用于从频域表示 X(ejω)X(e^{j\omega})X(ejω) 恢复时域序列 x[n]x[n]x[n]：

积分区间可以是任何长度为 2π2\pi2π 的区间，例如 [−π,π][-\pi, \pi][−π,π] 或 [0,2π][0, 2\pi][0,2π]。

存在条件：

一个序列 x[n]x[n]x[n] 存在 DTFT 的充分条件是该序列绝对可和，即：

∑n=−∞+∞∣x[n]∣<∞\sum_{n = -\infty}^{+\infty} |x[n]| < \inftyn=−∞∑+∞​∣x[n]∣<∞

对于不满足此条件的某些序列（如单位阶跃序列），其 DTFT 可能不收敛，或者需要在广义函数（如冲激函数）的意义下定义。有限能量序列（∑n=−∞+∞∣x[n]∣2<∞\sum_{n = -\infty}^{+\infty} |x[n]|^2 < \infty∑n=−∞+∞​∣x[n]∣2<∞）也存在 DTFT，但求和可能只在均方意义下收敛。

DTFT 的性质

DTFT 具有许多与 CTFT 类似的性质。设 x[n]⟷FX(ejω)x[n] \stackrel{\mathcal{F}}{\longleftrightarrow} X(e^{j\omega})x[n]⟷F​X(ejω) 和 y[n]⟷FY(ejω)y[n] \stackrel{\mathcal{F}}{\longleftrightarrow} Y(e^{j\omega})y[n]⟷F​Y(ejω)。

  1. 线性 (Linearity)：

    ax[n]+by[n]⟷FaX(ejω)+bY(ejω)ax[n] + by[n] \stackrel{\mathcal{F}}{\longleftrightarrow} aX(e^{j\omega}) + bY(e^{j\omega})ax[n]+by[n]⟷F​aX(ejω)+bY(ejω)

  2. 时移 (Time Shifting)：

    x[n−n0]⟷Fe−jωn0X(ejω)x[n-n_0] \stackrel{\mathcal{F}}{\longleftrightarrow} e^{-j\omega n_0}X(e^{j\omega})x[n−n0​]⟷F​e−jωn0​X(ejω)

    时域的整数延迟 n0n_0n0​ 对应于频域乘以一个线性相移因子 e−jωn0e^{-j\omega n_0}e−jωn0​。

  3. 频移 (Frequency Shifting / Modulation)：

    ejω0nx[n]⟷FX(ej(ω−ω0))e^{j\omega_0 n}x[n] \stackrel{\mathcal{F}}{\longleftrightarrow} X(e^{j(\omega - \omega_0)})ejω0​nx[n]⟷F​X(ej(ω−ω0​))

    时域乘以复指数 ejω0ne^{j\omega_0 n}ejω0​n 对应于频域的频谱搬移。

  4. 时间反转 (Time Reversal)：

    x[−n]⟷FX(e−jω)x[-n] \stackrel{\mathcal{F}}{\longleftrightarrow} X(e^{-j\omega})x[−n]⟷F​X(e−jω)

    如果 x[n]x[n]x[n] 是实序列，则 X(e−jω)=X∗(ejω)X(e^{-j\omega}) = X^*(e^{j\omega})X(e−jω)=X∗(ejω)。

  5. 共轭对称性 (Conjugate Symmetry for Real Sequences)：

    如果 x[n]x[n]x[n] 是实序列，则 X(ejω)=X∗(e−jω)X(e^{j\omega}) = X^*(e^{-j\omega})X(ejω)=X∗(e−jω)。 这意味着：

    • 幅度谱是偶对称的：∣X(ejω)∣=∣X(e−jω)∣|X(e^{j\omega})| = |X(e^{-j\omega})|∣X(ejω)∣=∣X(e−jω)∣
    • 相位谱是奇对称的：∠X(ejω)=−∠X(e−jω)\angle X(e^{j\omega}) = -\angle X(e^{-j\omega})∠X(ejω)=−∠X(e−jω)

  6. 卷积 (Convolution)：

    x[n]∗y[n]=∑k=−∞+∞x[k]y[n−k]⟷FX(ejω)Y(ejω)x[n] * y[n] = \sum_{k = -\infty}^{+\infty} x[k]y[n-k] \stackrel{\mathcal{F}}{\longleftrightarrow} X(e^{j\omega})Y(e^{j\omega})x[n]∗y[n]=∑k=−∞+∞​x[k]y[n−k]⟷F​X(ejω)Y(ejω)

    时域卷积对应频域相乘。这是分析离散时间 LTI 系统的核心性质。

  7. 相乘 (Multiplication / Windowing)：

    x[n]y[n]⟷F12πX(ejω)⊛Y(ejω)=12π∫2πX(ejθ)Y(ej(ω−θ))dθx[n]y[n] \stackrel{\mathcal{F}}{\longleftrightarrow} \frac{1}{2\pi}X(e^{j\omega}) \circledast Y(e^{j\omega}) = \frac{1}{2\pi}\int_{2\pi} X(e^{j\theta})Y(e^{j(\omega-\theta)})d\thetax[n]y[n]⟷F​2π1​X(ejω)⊛Y(ejω)=2π1​∫2π​X(ejθ)Y(ej(ω−θ))dθ

    时域相乘对应频域的周期卷积（并除以 2π2\pi2π）。

  8. 差分 (Differencing in Time Domain)：

    x[n]−x[n−1]⟷F(1−e−jω)X(ejω)x[n] - x[n-1] \stackrel{\mathcal{F}}{\longleftrightarrow} (1 - e^{-j\omega})X(e^{j\omega})x[n]−x[n−1]⟷F​(1−e−jω)X(ejω)

    这类似于连续时间中的微分，通常会增强高频成分。

    ∑k=−∞nx[k]⟷F11−e−jωX(ejω)+πX(ej0)∑k=−∞+∞δ(ω−2πk)\sum_{k=-\infty}^{n} x[k] \stackrel{\mathcal{F}}{\longleftrightarrow} \frac{1}{1-e^{-j\omega}}X(e^{j\omega}) + \pi X(e^{j0})\sum_{k=-\infty}^{+\infty}\delta(\omega - 2\pi k)∑k=−∞n​x[k]⟷F​1−e−jω1​X(ejω)+πX(ej0)∑k=−∞+∞​δ(ω−2πk)

    其中 X(ej0)=∑n=−∞+∞x[n]X(e^{j0}) = \sum_{n=-\infty}^{+\infty} x[n]X(ej0)=∑n=−∞+∞​x[n] 是序列的直流分量（所有样本之和）。如果 X(ej0)=0X(e^{j0})=0X(ej0)=0，则简化为 11−e−jωX(ejω)\frac{1}{1-e^{-j\omega}}X(e^{j\omega})1−e−jω1​X(ejω)。

  11. 帕塞瓦尔定理 (Parseval’s Relation / Energy Conservation)：

    ∑n=−∞+∞∣x[n]∣2=12π∫2π∣X(ejω)∣2dω\sum_{n=-\infty}^{+\infty} |x[n]|^2 = \frac{1}{2\pi}\int_{2\pi} |X(e^{j\omega})|^2 d\omega∑n=−∞+∞​∣x[n]∣2=2π1​∫2π​∣X(ejω)∣2dω

    这表示序列在时域的总能量等于其在频域的总能量（在一个周期内积分并除以 2π2\pi2π）。∣X(ejω)∣2|X(e^{j\omega})|^2∣X(ejω)∣2 被称为能量谱密度。

DTFT 与差分方程的联系

DTFT 是分析由线性常系数差分方程 (Linear Constant-Coefficient Difference Equations, LCCDEs) 描述的离散时间 LTI 系统的强大工具。

考虑一个 LTI 系统，其输入 x[n]x[n]x[n] 和输出 y[n]y[n]y[n] 由以下差分方程描述：

∑k=0Naky[n−k]=∑m=0Mbmx[n−m]\sum_{k=0}^{N} a_k y[n-k] = \sum_{m=0}^{M} b_m x[n-m]k=0∑N​ak​y[n−k]=m=0∑M​bm​x[n−m]

其中 aka_kak​ 和 bmb_mbm​ 是常数，通常假设 a0≠0a_0 \neq 0a0​=0（如果 a0=0a_0=0a0​=0，可以通过移位使新的 a0≠0a_0 \neq 0a0​=0）。通常将 a0a_0a0​ 归一化为 1。
