拉普拉斯变换

拉普拉斯变换是一种积分变换，它将一个实变量（通常是时间 ttt）的函数转换为一个复变量（通常是复频率 sss）的函数。它被广泛应用于工程和物理学中，特别是在电路分析、控制系统、信号处理等领域，用于简化微分方程和积分方程的求解。

双边拉普拉斯变换定义

对于一个信号 x(t)x(t)x(t)，其双边拉普拉斯变换 X(s)X(s)X(s) 定义为：

X(s)≜L{x(t)}=∫−∞+∞x(t)e−stdtX(s) \triangleq \mathcal{L}\{x(t)\} = \int_{-\infty}^{+\infty} x(t)e^{-st} dtX(s)≜L{x(t)}=∫−∞+∞​x(t)e−stdt

其中，sss 是一个复变量，通常表示为 s=σ+jωs = \sigma + j\omegas=σ+jω，这里：

  • σ\sigmaσ (sigma) 是实部，代表衰减或增长因子。
  • ω\omegaω (omega) 是虚部，代表角频率。

与傅里叶变换的关系

拉普拉斯变换可以看作是傅里叶变换的一种推广。回顾傅里叶变换的定义：

X(jω)=F{x(t)}=∫−∞+∞x(t)e−jωtdtX(j\omega) = \mathcal{F}\{x(t)\} = \int_{-\infty}^{+\infty} x(t)e^{-j\omega t} dtX(jω)=F{x(t)}=∫−∞+∞​x(t)e−jωtdt

如果我们将拉普拉斯变换中的 sss 替换为 jωj\omegajω（即令 σ=0\sigma = 0σ=0），则拉普拉斯变换的形式与傅里叶变换相同。

更一般地，我们可以将拉普拉斯变换 X(s)X(s)X(s) 看作是信号 x(t)e−σtx(t)e^{-\sigma t}x(t)e−σt 的傅里叶变换：

X(s)=X(σ+jω)=∫−∞+∞x(t)e−(σ+jω)tdt=∫−∞+∞[x(t)e−σt]e−jωtdtL{x(t)}=F{x(t)e−σt}\begin{aligned} X(s) &= X(\sigma + j\omega) = \int_{-\infty}^{+\infty} x(t)e^{-(\sigma + j\omega)t} dt \\ &= \int_{-\infty}^{+\infty} [x(t)e^{-\sigma t}]e^{-j\omega t} dt \\ \mathcal{L}\{x(t)\} &= \mathcal{F}\{x(t)e^{-\sigma t}\} \end{aligned}X(s)L{x(t)}​=X(σ+jω)=∫−∞+∞​x(t)e−(σ+jω)tdt=∫−∞+∞​[x(t)e−σt]e−jωtdt=F{x(t)e−σt}​

这里的 e−σte^{-\sigma t}e−σt 项是一个衰减因子（如果 σ>0\sigma > 0σ>0）或增长因子（如果 σ<0\sigma < 0σ<0）。这个因子的引入使得许多傅里叶变换不存在的信号（例如 eatu(t)e^{at}u(t)eatu(t) 当 a>0a>0a>0 时）也可能具有拉普拉斯变换，只要能找到合适的 σ\sigmaσ 值使得积分收敛。

收敛域 (Region of Convergence, ROC)

由于积分限是从 −∞-\infty−∞ 到 +∞+\infty+∞，并且存在 e−ste^{-st}e−st 项，拉普拉斯变换的积分不一定对所有的复数 sss 都收敛。使积分收敛的 sss 值的集合称为收敛域 (ROC)。

ROC 是 sss 平面上的一个区域，它对于确定拉普拉斯变换的唯一性以及分析系统的性质（如稳定性、因果性）至关重要。对于一个给定的 X(s)X(s)X(s)，可能存在多个不同的时间函数 x(t)x(t)x(t) 对应它，但如果同时指定了 X(s)X(s)X(s) 及其 ROC，则 x(t)x(t)x(t) 是唯一的。

ROC 的一些性质：

  1. ROC 是 sss 平面上的带状区域，其边界平行于 jωj\omegajω 轴。
  2. 如果 x(t)x(t)x(t) 是有限持续的，并且绝对可积，则 ROC 是整个 sss 平面。
  3. 如果 x(t)x(t)x(t) 是右边信号（即当 t<T1t < T_1t<T1​ 时 x(t)=0x(t)=0x(t)=0），则 ROC 是一个右半平面，形如 Re{s}>σmax\text{Re}\{s\} > \sigma_{max}Re{s}>σmax​。
  4. 如果 x(t)x(t)x(t) 是左边信号（即当 t>T2t > T_2t>T2​ 时 x(t)=0x(t)=0x(t)=0），则 ROC 是一个左半平面，形如 Re{s}<σmin\text{Re}\{s\} < \sigma_{min}Re{s}<σmin​。
  5. 如果 x(t)x(t)x(t) 是双边信号，则 ROC 可能是一个垂直带状区域 σ1<Re{s}<σ2\sigma_1 < \text{Re}\{s\} < \sigma_2σ1​<Re{s}<σ2​，或者不存在。
  6. ROC 不包含任何极点 (poles)。

逆拉普拉斯变换 (Inverse Laplace Transform)

逆拉普拉斯变换用于从频域表示 X(s)X(s)X(s) 恢复时域信号 x(t)x(t)x(t)。其定义为：

x(t)=L−1{X(s)}=12πj∫σ−j∞σ+j∞X(s)estdsx(t) = \mathcal{L}^{-1}\{X(s)\} = \frac{1}{2\pi j}\int_{\sigma - j\infty}^{\sigma + j\infty}X(s)e^{st} dsx(t)=L−1{X(s)}=2πj1​∫σ−j∞σ+j∞​X(s)estds

这个积分是在 sss 平面上的复平面积分，积分路径是一条平行于 jωj\omegajω 轴的直线 Re{s}=σ\text{Re}\{s\} = \sigmaRe{s}=σ，其中 σ\sigmaσ 必须位于 X(s)X(s)X(s) 的收敛域内。在实际应用中，通常使用查表法和部分分式展开法来求逆变换，而不是直接计算这个复积分。

单边拉普拉斯变换 (Unilateral Laplace Transform)

单边拉普拉斯变换是双边拉普拉斯变换的一个特例，它主要用于分析因果信号和系统，特别是那些在 t<0t<0t<0 时值为零的信号。它在求解带有初始条件的线性常系数微分方程时非常有用。

单边拉普拉斯变换定义

对于一个信号 x(t)x(t)x(t)，其单边拉普拉斯变换 X(s)X(s)X(s) 定义为：

X(s)≜Lu{x(t)}=∫0−+∞x(t)e−stdtX(s) \triangleq \mathcal{L}_u\{x(t)\} = \int_{0^{-}}^{+\infty} x(t)e^{-st} dtX(s)≜Lu​{x(t)}=∫0−+∞​x(t)e−stdt

积分下限通常写为 0−0^{-}0−，表示积分包含 t=0t=0t=0 处的任何冲击或不连续性。如果 x(t)x(t)x(t) 在 t=0t=0t=0 处没有冲击或更高阶的奇异性，下限也可以是 000。

主要特点和应用

  1. 处理因果信号：由于积分从 0−0^{-}0− 开始，单边拉普拉斯变换自动忽略了 t<0t<0t<0 时的信号行为。因此，它天然适用于因果信号（即对于 t<0t<0t<0，x(t)=0x(t)=0x(t)=0 的信号）。对于因果信号，其单边和双边拉普拉斯变换是相同的。
  2. 求解微分方程：单边拉普拉斯变换的一个关键优势在于它能方便地处理线性常系数微分方程的初始条件。其时域微分性质包含了初始条件项：
    • Lu{dx(t)dt}=sX(s)−x(0−)\mathcal{L}_u\left\{\frac{dx(t)}{dt}\right\} = sX(s) - x(0^{-})Lu​{dtdx(t)​}=sX(s)−x(0−)
    • Lu{d2x(t)dt2}=s2X(s)−sx(0−)−x′(0−)\mathcal{L}_u\left\{\frac{d^2x(t)}{dt^2}\right\} = s^2X(s) - sx(0^{-}) - x'(0^{-})Lu​{dt2d2x(t)​}=s2X(s)−sx(0−)−x′(0−) 以此类推。这使得将微分方程转换为代数方程时，初始条件可以直接代入。
  3. 系统分析：在分析因果线性时不变 (LTI) 系统时，如果输入也是因果的，单边拉普拉斯变换非常方便。

单边拉普拉斯变换是将线性常系数微分方程 (LCCDE) 转换为代数方程的强大工具，从而简化求解过程，尤其是当初始条件已知时。

求解步骤如下：

  1. 对微分方程两边取单边拉普拉斯变换： 对微分方程中的每一项应用单边拉普拉斯变换。利用其微分性质，例如：

    • Lu{dy(t)dt}=sY(s)−y(0−)\mathcal{L}_u\left\{\frac{dy(t)}{dt}\right\} = sY(s) - y(0^{-})Lu​{dtdy(t)​}=sY(s)−y(0−)
    • Lu{d2y(t)dt2}=s2Y(s)−sy(0−)−y′(0−)\mathcal{L}_u\left\{\frac{d^2y(t)}{dt^2}\right\} = s^2Y(s) - sy(0^{-}) - y'(0^{-})Lu​{dt2d2y(t)​}=s2Y(s)−sy(0−)−y′(0−) 其中 Y(s)=Lu{y(t)}Y(s) = \mathcal{L}_u\{y(t)\}Y(s)=Lu​{y(t)} 是输出信号 y(t)y(t)y(t) 的拉普拉斯变换，y(0−)y(0^{-})y(0−), y′(0−)y'(0^{-})y′(0−) 等是 t=0−t=0^{-}t=0− 时的初始条件。对于输入信号 x(t)x(t)x(t)，其拉普拉斯变换为 X(s)X(s)X(s)。

  2. 代入初始条件： 将给定的初始条件代入变换后的方程中。

  3. 求解 Y(s)Y(s)Y(s)： 经过变换和代入初始条件后，原微分方程会变成一个关于 Y(s)Y(s)Y(s) 和 X(s)X(s)X(s) 的代数方程。求解这个代数方程，得到 Y(s)Y(s)Y(s) 的表达式，通常形式为：

    Y(s)=N(s)D(s)Y(s) = \frac{N(s)}{D(s)}Y(s)=D(s)N(s)​

    其中 N(s)N(s)N(s) 和 D(s)D(s)D(s) 是关于 sss 的多项式。Y(s)Y(s)Y(s) 通常包含两部分：一部分由输入 X(s)X(s)X(s) 产生（零状态响应），另一部分由初始条件产生（零输入响应）。

  4. 对 Y(s)Y(s)Y(s) 取逆拉普拉斯变换得到 y(t)y(t)y(t)： 使用部分分式展开法和拉普拉斯变换表，求 Y(s)Y(s)Y(s) 的逆拉普拉斯变换，得到时域解 y(t)y(t)y(t)。

    y(t)=Lu−1{Y(s)}y(t) = \mathcal{L}_u^{-1}\{Y(s)\}y(t)=Lu−1​{Y(s)}
示例

考虑以下一阶线性常系数微分方程：

dy(t)dt+2y(t)=x(t)\frac{dy(t)}{dt} + 2y(t) = x(t)dtdy(t)​+2y(t)=x(t)

假设输入 x(t)=u(t)x(t) = u(t)x(t)=u(t) (单位阶跃函数)，初始条件为 y(0−)=1y(0^{-}) = 1y(0−)=1。

  1. 取拉普拉斯变换： 对等式两边取单边拉普拉斯变换： Lu{dy(t)dt}+2Lu{y(t)}=Lu{x(t)}\mathcal{L}_u\left\{\frac{dy(t)}{dt}\right\} + 2\mathcal{L}_u\{y(t)\} = \mathcal{L}_u\{x(t)\}Lu​{dtdy(t)​}+2Lu​{y(t)}=Lu​{x(t)} [sY(s)−y(0−)]+2Y(s)=X(s)[sY(s) - y(0^{-})] + 2Y(s) = X(s)[sY(s)−y(0−)]+2Y(s)=X(s)

  2. 代入初始条件和输入： 已知 y(0−)=1y(0^{-}) = 1y(0−)=1，x(t)=u(t)  ⟹  X(s)=1sx(t) = u(t) \implies X(s) = \frac{1}{s}x(t)=u(t)⟹X(s)=s1​。 [sY(s)−1]+2Y(s)=1s[sY(s) - 1] + 2Y(s) = \frac{1}{s}[sY(s)−1]+2Y(s)=s1​

  3. 求解 Y(s)Y(s)Y(s)： Y(s)(s+2)−1=1sY(s)(s+2) - 1 = \frac{1}{s}Y(s)(s+2)−1=s1​ Y(s)(s+2)=1+1s=s+1sY(s)(s+2) = 1 + \frac{1}{s} = \frac{s+1}{s}Y(s)(s+2)=1+s1​=ss+1​ Y(s)=s+1s(s+2)Y(s) = \frac{s+1}{s(s+2)}Y(s)=s(s+2)s+1​

  4. 取逆拉普拉斯变换： 使用部分分式展开： s+1s(s+2)=As+Bs+2\frac{s+1}{s(s+2)} = \frac{A}{s} + \frac{B}{s+2}s(s+2)s+1​=sA​+s+2B​ A=s+1s+2∣s=0=12A = \left.\frac{s+1}{s+2}\right|_{s=0} = \frac{1}{2}A=s+2s+1​​s=0​=21​ B=s+1s∣s=−2=−1−2=12B = \left.\frac{s+1}{s}\right|_{s=-2} = \frac{-1}{-2} = \frac{1}{2}B=ss+1​​s=−2​=−2−1​=21​ 所以，Y(s)=1/2s+1/2s+2Y(s) = \frac{1/2}{s} + \frac{1/2}{s+2}Y(s)=s1/2​+s+21/2​ 对其取逆拉普拉斯变换： y(t)=12u(t)+12e−2tu(t)y(t) = \frac{1}{2}u(t) + \frac{1}{2}e^{-2t}u(t)y(t)=21​u(t)+21​e−2tu(t) for t≥0t \ge 0t≥0.

这种方法系统地将求解微分方程的问题转化为代数运算和查表，大大简化了求解过程。

总结：

  • 双边拉普拉斯变换 更具一般性，可以分析各种类型的信号，其收敛域对于理解信号特性至关重要。
  • 单边拉普拉斯变换 是双边变换在 t≥0t \ge 0t≥0 区间上的简化版本，特别适用于处理因果系统和带有初始条件的微分方程问题。在许多工程教科书中，如果没有特别指明，“拉普拉斯变换” 通常指的是单边拉普拉斯变换，因为它更贴近实际物理系统的分析。
